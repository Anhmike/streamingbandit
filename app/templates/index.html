Welcome to Streampy!

This is the homepage for the Streampy server hosted by the N-th Iteration Lab. Solving (bandit) decision problems in data streams in production environment is challenging. With this Python module we are trying to develop a framework for streaming/online solutions to bandit problems which can be used to experiment with novel solutions in production environments.
