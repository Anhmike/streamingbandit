Admin API¶

POST (?i)/admin/exp/add.json¶

Create a new experiment

Requires:

A secure cookie obtained by logging in.
Parameters:
  • name (string) – Name of the experiment.
  • getcontext (string) – String of python code for get context code.
  • getaction (string) – String of python code for get action code.
  • getreward (string) – String of python code for get reward code.
  • setreward (string) – String of python code for set reward code.
  • advice_id (bool) – Bool indicating whether advice_id’s are used.
  • hourly (bool) – Bool indicating whether the state of Theta should be stored hourly.
  • advice_id – Bool indicating whether the getAdvice and setReward calls should return an advice_id.
  • delta_days (int) – If advice_id is True, supply this to give the number of days that an advice_id should be stored.
  • default_reward (dict) – If advice_id is True, supply this to give the default reward for advice_id’s that are over their delta_days limit.
Returns:

A JSON of the form: { id : the assigned experiment id,

name : the name of the experiment (checked for duplicates), error : (optional) error message }
Raises 401:

If user is not logged in or if there is no secure cookie available.
