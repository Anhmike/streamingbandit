About StreamingBandit¶

Contributing to the libraries¶

We gladly accept any contribution to expanding the libraries of StreamingBandit. The libraries that are currently written are all tested and have an agent that functions with them. If you would like to contribute by writing a part of the library, look at the following points:

  • Write a Base class according to the skeleton class __strmBase.
  • Write a default example of a use-case of the class in the defaults folder.
  • Write an agent in the utils folder that uses the class and the default example.
